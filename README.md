# Eclipse IDE for Java Developers: Beginner to Advanced

Hands-on Java projects and examples for mastering **Eclipse IDE**, from fundamental project management and editor productivity to Maven, debugging, Spring Boot, Git, GitHub and AI-assisted development.

This repository is the source-code companion for the course of the same name.

**Course URL:** [Coming soon][course-url]

> This repository is being developed alongside the course. New packages, examples and lesson references are added progressively.

---

## What You Will Learn

- Install and configure Eclipse IDE for Java development
- Create, open, import, export and manage Java projects
- Use Content Assist and configure automatic code completion
- Use built-in and custom Eclipse code templates
- Apply Quick Fix and Quick Assist with `Ctrl+1`
- Format, generate and refactor Java code
- Navigate and search large Java projects efficiently
- Manage Maven projects and dependencies in Eclipse
- Debug Java applications from basic to advanced levels
- Use conditional breakpoints, watchpoints and exception breakpoints
- Debug Spring Boot and REST applications
- Use AI-assisted coding tools inside Eclipse
- Work with Git and GitHub through Eclipse

## Quick Fix and Quick Assist Demo

Class: `org.eclipsefeaturesdemo.editing.QuickFixQuickAssistDemo`

This lesson demonstrates how Eclipse uses the same shortcut — `Ctrl+1` — for two related features:

- **Quick Fix** corrects detected errors and warnings.
- **Quick Assist** transforms valid Java code.

The demonstration includes:

- Adding missing imports
- Creating a missing method
- Assigning an expression to a local variable
- Reviewing proposals based on the current cursor position
